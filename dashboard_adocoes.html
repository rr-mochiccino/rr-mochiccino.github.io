Centro de Adoções

No dashboard de adoções foi utilizado a base de dados da cidade de Austin para criar uma visão interativa que mostra a quantidade total de animais disponíveis para adoção, as diversas categorias que representam diferentes espécies e grupos de animais, além da média e quantidade de adoções recentes. Este resumo proporciona uma compreensão rápida do impacto que a comunidade de adotantes está tendo.

Detalhes por Categoria: Ao navegar para a segunda página do dashboard, você terá acesso a uma análise mais detalhada de cada categoria de animais disponíveis para adoção. Aqui, destacamos informações cruciais, como a quantidade de castrações realizadas, raças presentes, variedade de cores, média da idade dos animais e os desfechos de cada adoção.
